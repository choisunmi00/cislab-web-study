# 📚 Resources

스터디에 필요한 참고 자료와 템플릿을 관리하는 폴더입니다.

## 📁 폴더 구조

- `references/` - 참고 자료, 문서, 링크 모음
- `templates/` - 코드 템플릿, 보일러플레이트 코드

## 🎯 목적

- 스터디 과정에서 필요한 모든 참고 자료 중앙 관리
- 재사용 가능한 코드 템플릿 제공
- 유용한 링크와 문서 아카이빙

## 📖 포함 내용

### References

- 기술 문서 및 가이드
- 유용한 웹사이트 링크
- 추천 도서 및 강의 정보
- 개발 도구 및 라이브러리 정보

### Templates

- 프로젝트 초기 설정 템플릿
- 공통 컴포넌트 템플릿
- API 구조 보일러플레이트
- 코딩 컨벤션 및 스타일 가이드
