# 📄 Templates

재사용 가능한 코드 템플릿과 보일러플레이트를 모아놓은 폴더입니다.

## 📋 포함 내용

- **프로젝트 초기 설정**

  - React 프로젝트 초기 구조
  - FastAPI 프로젝트 템플릿
  - 환경 설정 파일들

- **공통 컴포넌트**

  - 재사용 가능한 React 컴포넌트
  - 공통 CSS 스타일
  - 레이아웃 템플릿

- **API 구조**

  - REST API 보일러플레이트
  - 데이터베이스 모델 템플릿
  - 인증/인가 구조

- **설정 파일**
  - 코딩 컨벤션 가이드
  - ESLint, Prettier 설정
  - 환경 변수 템플릿

## 🎯 사용 방법
