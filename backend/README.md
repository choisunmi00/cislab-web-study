# 🔧 Backend

FastAPI를 사용한 백엔드 개발 학습 자료와 코드를 관리하는 폴더입니다.

## 📁 폴더 구조

- `lectures/` - 백엔드 강의 자료 및 실습 코드
- `assignments/` - 백엔드 과제 및 솔루션
- `members/` - 백엔드 팀원들의 개인 학습 자료

## 🎯 학습 목표

- FastAPI 서버 구축 및 API 개발
- 데이터베이스 연동 및 ORM 활용
- 인증/인가 시스템 구현
- 외부 API 연동 및 LLM 기능 개발
