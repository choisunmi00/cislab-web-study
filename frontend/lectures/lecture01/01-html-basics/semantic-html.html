시멘틱 HTML이란?

정의

시멘틱 HTML은 HTML 태그가 가진 의미를 살려 구조적으로 마크업하는 방식입니다. 각 태그가 담고 있는 내용의 의미를 명확히 전달합니다.

예시: <div> 대신 <article>, <section>, <nav> 등을 사용

주요 시멘틱 요소들

<header>

페이지나 섹션의 헤더를 나타냅니다. 보통 제목, 로고, 네비게이션 등을 포함합니다.

<header> <h1>사이트 제목</h1> <nav>네비게이션</nav> </header>

<nav>

네비게이션 링크들을 그룹화하는 요소입니다.

<nav> <ul> <li><a href="#home">홈</a></li> <li><a href="#about">소개</a></li> </ul> </nav>

<main>

페이지의 주요 콘텐츠를 나타냅니다. 페이지당 하나만 사용해야 합니다.

<main> <!-- 페이지의 주요 내용 --> </main>

<section>

문서의 구역을 나타냅니다. 관련된 내용을 그룹화할 때 사용합니다.

<section> <h2>섹션 제목</h2> <p>섹션 내용</p> </section>

<article>

독립적으로 배포 가능한 콘텐츠를 나타냅니다. 블로그 포스트, 뉴스 기사 등에 사용합니다.

<article> <h2>기사 제목</h2> <p>기사 내용</p> </article>

<aside>

주요 내용과 관련되지만 독립적인 내용을 나타냅니다. 사이드바, 광고 등에 사용합니다.

<aside> <h3>관련 링크</h3> <ul> <li><a href="#">관련 기사 1</a></li> </ul> </aside>

<footer>

페이지나 섹션의 푸터를 나타냅니다. 저작권, 연락처 등을 포함합니다.

<footer> <p>© 2024 회사명. 모든 권리 보유.</p> </footer>

비교 예시

❌ 시멘틱하지 않은 방식

<div class="header"> <div class="title">제목</div> <div class="nav">네비게이션</div> </div> <div class="content"> <div class="article">기사 내용</div> </div> <div class="footer">푸터</div>

✅ 시멘틱한 방식

<header> <h1>제목</h1> <nav>네비게이션</nav> </header> <main> <article>기사 내용</article> </main> <footer>푸터</footer>
