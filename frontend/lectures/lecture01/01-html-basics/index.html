3. 자주 사용하는 태그들

div - 블록 컨테이너

이것은 div 태그입니다. 블록 레벨 요소로 전체 너비를 차지합니다.

span - 인라인 컨테이너

이 문단에서 이 부분만 다른 스타일을 적용했습니다.

제목 태그 (h1~h6)

h1 - 가장 큰 제목

h2 - 두 번째 제목

h3 - 세 번째 제목

h4 - 네 번째 제목

h5 - 다섯 번째 제목
h6 - 가장 작은 제목

목록 태그 (ul, ol, li)

순서 없는 목록 (ul)

  • 첫 번째 항목
  • 두 번째 항목
  • 세 번째 항목

순서 있는 목록 (ol)

  1. 첫 번째 단계
  2. 두 번째 단계
  3. 세 번째 단계

문단과 링크 (p, a)

이것은 문단(paragraph) 태그입니다. 텍스트 내용을 담는 기본 태그입니다.

클릭 가능한 링크를 포함할 수 있습니다.

4. class와 id의 의미

class 속성

class는 여러 요소에 동일한 스타일을 적용할 때 사용합니다.

같은 class를 여러 요소에 적용 가능합니다.

id 속성

id는 페이지에서 고유한 요소를 식별할 때 사용합니다.

id는 한 페이지에서 유일해야 합니다.

5. 부모, 자식, 형제 관계

부모 요소 (Parent)

첫 번째 자식 요소 (Child)
두 번째 자식 요소 (Child) - 첫 번째와 형제 관계
세 번째 자식 요소 (Child) - 다른 자식들과 형제 관계
