CSS 선택자와 Display 속성 학습

1. 태그 선택자 (Tag Selector)

이 문단은 태그 선택자로 스타일이 적용되었습니다.

모든 p 태그에 동일한 스타일이 적용됩니다.

p { color: #666; } h2 { color: #2196F3; border-bottom: 2px solid #2196F3; }

2. 클래스 선택자 (Class Selector)

이 문단은 .highlight 클래스가 적용되었습니다.

3. ID 선택자 (ID Selector)

이 요소는 고유한 ID를 가집니다.

특별한 제목

ID는 페이지에서 유일해야 합니다.

#unique-element { background-color: #e8f5e8; padding: 10px; border: 2px solid #4CAF50; } #special-title { color: #ff6b6b; font-size: 24px; }

4. Pseudo 선택자 (Pseudo Selector)

마우스를 올려보세요 (hover 효과)

링크 상태별 스타일

링크를 클릭해보세요 (visited, active 상태 확인)

.hover-effect:hover { background-color: #ffeb3b; cursor: pointer; } a:link { color: blue; } a:visited { color: purple; } a:hover { color: red; } a:active { color: orange; }

5.2 Display None/Block 토글

이 요소는 보였다가 숨겨집니다.
.block-example { display: block; /* 전체 너비 차지, 세로로 배치 */ } .inline-example { display: inline; /* 필요한 만큼의 너비, 가로로 배치 */ } .inline-block-example { display: inline-block; /* 가로 배치 + 크기 조절 가능 */ } .none-example { display: none; /* 완전히 숨김 */ }
