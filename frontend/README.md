# ⚛️ Frontend

React를 사용한 프론트엔드 개발 학습 자료와 코드를 관리하는 폴더입니다.

## 📁 폴더 구조

- `lectures/` - 프론트엔드 강의 자료 및 실습 코드
- `assignments/` - 프론트엔드 과제 및 솔루션
- `members/` - 프론트엔드 팀원들의 개인 학습 자료

## 🎯 학습 목표

- React 컴포넌트 개발 및 Hooks 활용
- 상태관리 및 라우팅 구현
- 사용자 인터페이스 디자인 및 UX 개선
- 백엔드 API와의 통신 및 데이터 처리
